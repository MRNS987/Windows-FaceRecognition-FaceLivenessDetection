1. Install things in dependency folder
 When install python, there's checkbox of adding python into system path in first screen, tick it.
2. Enter the demo folder which you want to test (Flask, Gradio, Video Surveillance Demo)
3. In cmd, python -m pip install -r requirements.txt
4. In demo folder, python app.py
 It will show HWID and share to us.
5. Replace content of license.txt in ./engine folder with license we provide.
6. Run python app.py
